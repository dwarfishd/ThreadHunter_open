# ThreadHunter

Local-first система мониторинга Telegram-каналов, которая превращает поток неструктурированных объявлений в структурированную базу данных. Проект автоматически извлекает контакты, классифицирует публикации по десяткам бизнес-критериев и помогает быстро находить релевантных поставщиков и подрядчиков без ручного просмотра сотен сообщений.

## Требования

- Python 3.10+
- Telegram API credentials (бесплатно)

## Установка

```bash
pip install -e .
```

Или скачайте бинарник из GitHub Releases (macOS / Windows / Linux).

## Настройка API

1. Перейдите на https://my.telegram.org/apps
2. Войдите с номером телефона
3. Создайте приложение — получите **API_ID** и **API_HASH**

## Использование

```bash
# 1. Инициализация (создаёт БД и .env)
th init

# 2. Впишите API_ID и API_HASH в файл .env

# 3. Авторизация в Telegram (телефон → код → сессия)
th auth

# 4. Добавить канал для мониторинга
th add-channel @channel_name

# 5. Скачать посты, извлечь контакты, классифицировать теги
th parse

# 6. Посмотреть состояние базы
th status
```

## Структура

```
src/threadhunter/
├── cli.py           # CLI-команды (Typer)
├── config.py        # Настройки (.env → Pydantic)
├── pipeline.py      # Оркестрация: контакты + теги
├── channels.py      # CRUD каналов
├── posts.py         # CRUD постов
├── db/              # SQLite: схема, модели, подключение
├── extract/         # Извлечение контактов и тегов
└── telegram/        # Telethon: авторизация, клиент
```

## Команды

| Команда | Описание |
|---|---|
| `th init` | Создать БД и `.env` |
| `th auth` | Авторизация в Telegram |
| `th add-channel @name` | Добавить канал |
| `th parse` | Скачать посты и обработать |
| `th status` | Состояние базы |
| `th export` | Экспорт (в разработке) |
