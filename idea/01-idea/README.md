## Idea Canvas

**Описание:** Приложение для парсинга объявлений из Telegram-каналов. Извлекает контакты, текст объявления и дополнительную информацию. Объявления сортируются по тегам.

**Фокус:** Швейные производства в Бишкеке и России.

## Источники
- Только Telegram-каналы

## Теги (сортировка объявлений)
- **Трекер (локация)** — привязка к местности (город, регион)
- **Ассортимент** — что шьют, номенклатура отшиваемой продукции
- *Остальные теги будут добавлены позже*
