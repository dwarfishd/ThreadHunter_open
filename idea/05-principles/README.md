## Принципы продукта

Четыре принципа, которые определяют каждое продуктовое и техническое решение:

- [[05-principles/local-first|local-first]] — Данные и обработка локально. Нет облака, нет сервера, нет зависимости.
- [[05-principles/proactive|proactive]] — Система парсит сама по расписанию. Пользователь приходит к готовым данным.
- [[05-principles/action-oriented|action-oriented]] — Минимум шагов от данных к действию. Не смотреть, а связаться.
- [[05-principles/one-person-company|one-person-company]] — Один человек управляет всем циклом. Без IT, без команды, без настроек.
