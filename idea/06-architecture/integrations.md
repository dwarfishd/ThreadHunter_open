# Integrations

## Внешние интеграции

### MVP: Telegram API
- **Telethon** (Python) или **GramJS** (Node.js) — библиотека для работы с Telegram MTProto API
- Чтение сообщений из каналов
- Извлечение метаданных постов
- Единственная внешняя интеграция в MVP

### Future: Экспорт
- **CSV/Excel** — экспорт базы контактов для использования в внешних инструментах
- **JSON** — для программной интеграции

### Future: CRM
- **Отправка контактов в CRM** — HubSpot, amoCRM, Bitrix24
- **Обратная связь** — отметка «связался», «отказ», «в работе» синхронизируется обратно
- **Двусторонняя синхронизация** — статусы из CRM влияют на отображение в продукте

### НЕ планируется
- Интеграция с мессенджерами для рассылки (продукт не спамит)
- Интеграция с маркетплейсами
- Интеграция с email-сервисами

## Принцип интеграций

- **MVP = 1 интеграция** (Telegram). Всё остальное — потом.
- **Future интеграции — через экспорт файлов**, не через API. Проще, надёжнее, меньше зависимостей.
- **CRM — только когда продукт доказал ценность**. До этого — экспорт в CSV достаточно.
