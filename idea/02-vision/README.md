## Видение продукта

Автоматизированный парсер Telegram-каналов для швейной ниши. Собирает, структурирует и классифицирует объявления по тегам (локация, ассортимент). Пользователь получает готовую базу контактов для работы: тендеры, сотрудничество, предложение услуг продвижения.

## Ключевые элементы
- Периодический парсинг (день/неделя/месяц) — см. [[06-architecture/triggers|Triggers]]
- Автоматическая сортировка по тегам — см. [[06-architecture/agent-loop|Agent Loop]]
- Готовая база для outreach и сотрудничества — см. [[05-principles/action-oriented|Action Oriented]]
