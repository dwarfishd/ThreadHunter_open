# Jobs to be Done

## Основная задача

Менеджер использует продукт для сбора базы данных потенциальных подрядчиков и клиентов из Telegram-каналов.

## Контекст
- Нужны подрядчики для тендеров
- Нужны клиенты для предложения услуг
- Требуется постоянный мониторинг рынка швейных производств

## Результат
Структурированная база контактов с информацией о локации и ассортименте, готовая для outreach и работы.
