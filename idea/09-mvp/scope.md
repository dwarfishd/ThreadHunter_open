# Scope

## MVP включает:

1. **Автоматический сбор данных** из Telegram-каналов
2. **Автоматическая сортировка** по тегам (локация, ассортимент)
3. **Начальная форма аналитики**

## Собираемые поля:

- Контакты (телефон, имя)
- Текст объявления
- Локация (город, регион)
- Ассортимент (что шьют)
- Дата публикации
- Ссылка на пост/канал (источник)
- Тип предложения (ищут подрядчика / предлагают услуги / ищут сотрудников)
- Объёмы/сроки
- Цены/расценки
- Наличие фото/вложений
- Статус (активно/закрыто)

*Принцип: собираем всё, потом отфильтруем ненужное.*

## НЕ входит в MVP:
- Outreach/рассылки
- Продвинутая аналитика
- Интеграции с внешними системами
- Ручной ввод данных
