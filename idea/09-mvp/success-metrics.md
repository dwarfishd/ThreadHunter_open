# Success Metrics

## Главный критерий успеха

**Успешно спарсен весь Telegram-канал** — все объявления из канала собраны, структурированы и сохранены в базу без потерь данных.

## Дополнительные метрики
- Полнота сбора: все посты из канала присутствуют в базе
- Качество извлечения: контакты, тексты, теги — корректно распознаны
- Автоматическая сортировка: объявления корректно распределены по тегам
