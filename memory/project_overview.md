ThreadHunter — CLI-утилита на Python 3.10+ для парсинга объявлений из Telegram-каналов, извлечения контактов и AI-классификации по тегам (локация, ассортимент). Целевая аудитория — менеджеры швейных производств в Бишкеке и России. Стек: Telethon (Telegram API), SQLite (локальная БД), Typer (CLI), spaCy/Ollama (AI-классификация), Pydantic (валидация). Принцип Local First: все данные локально, один SQLite-файл, ноль серверов. Деплой через GitHub Releases + PyInstaller.

**Why:** Архитектура определяет Local First — данные не уходят наружу, пользователь сам защищает свою машину.

**How to apply:** При принятии решений отдавать предпочтение локальным решениям перед облачными. Не предлагать серверные компоненты или облачные API без явного запроса.
